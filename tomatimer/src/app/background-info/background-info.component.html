Background Info

What is Tomatimer?

Tomatimer is a website intended to improve general productivity by helping the user manage their work / break balance through repeating timers.

The website is mainly intended for use with the pomodoro technique, which is a time management system in which the users work for 25 minutes and then take 5-minute breaks, repeated for four cycles. This website is equipped to deal with any amount of work, break lengths depending on the individual users need.

In the future, this website is to have an account system, that helps track the user's total work time daily, helping the user manages how productive they are over a long period of time.
